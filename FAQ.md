# அடிக்கடி கேட்கப்படும் கேள்விகள் (FAQ)

---

### 1. இந்தக் கருவி என்ன செய்கிறது?

பழைய தமிழ் PDF கோப்புகளில் உள்ள உரையை, நவீன Unicode (UTF-8) தமிழாக மாற்றுகிறது.
1990கள் மற்றும் 2000களின் தொடக்கத்தில் வெளியான அரசு மற்றும் பல்கலைக்கழக வெளியீடுகள் பலவும் **TSCII** என்ற பழைய குறியாக்கத்தில் (encoding) இருக்கும். இந்தக் கருவி அந்தக் குறியாக்கத்தை Unicode-ஆக மாற்றி, படிக்கக்கூடிய Markdown கோப்பாக உருவாக்குகிறது.

---

### 2. எனது PDF-ஐ இந்தக் கருவி படிக்க முடியுமா?

பின்வரும் அட்டவணை உதவும்:

| PDF வகை | பொருந்துமா? |
| ------- | ---------- |
| TSCII குறியாக்கம் (1990s–2000s அரசு/பல்கலைக்கழக வெளியீடுகள்) | ✅ ஆம் |
| Unicode தமிழ் PDF (2010-க்கு பிறகு வெளியானவை) | ❌ இல்லை — உரை ஏற்கனவே படிக்கக்கூடியது |
| ஸ்கேன் செய்யப்பட்ட படம் (image-based PDF) | ❌ இல்லை — OCR தேவை |

---

### 3. எனது PDF TSCII-ல் இருக்கிறதா என்று எப்படி தெரிந்துகொள்வது?

PDF-ல் இருந்து சிறிது உரையை நகலெடுத்து (copy) ஒரு text editor-ல் ஒட்டுங்கள் (paste). தமிழ் எழுத்துக்களுக்கு பதிலாக ஆங்கில எழுத்துக்கள் அல்லது சின்னங்கள் (symbols) தெரிந்தால், அந்த PDF TSCII குறியாக்கத்தில் உள்ளது — இந்தக் கருவி உதவும்.

---

### 4. இந்தக் கருவியை எப்படி நிறுவுவது (install)?

Python 3.10 அல்லது அதற்கு மேல் இருக்க வேண்டும். பிறகு பின்வரும் கட்டளையை இயக்கவும்:

```bash
pip install pypdf open-tamil
```

அல்லது:

```bash
pip install -r requirements.txt
```

---

### 5. கருவியை எப்படி இயக்குவது?

உங்கள் சொந்த PDF கோப்புக்கு:

```bash
python3 tamil-converter.py உங்கள்_கோப்பு.pdf வெளியீடு.md
```

உதாரணம்:

```bash
python3 tamil-converter.py docs/அகராதி.pdf output/அகராதி.md
```

---

### 6. வெளியீடு (output) எந்த வடிவத்தில் இருக்கும்?

கருவி `ஆங்கிலம் - தமிழ்` என்ற வடிவில் உள்ள வரிகளை கண்டறிந்து, அவற்றை இரண்டு நெடுவரிசை (two-column) Markdown அட்டவணையாக எழுதும்:

```markdown
| ஆங்கிலம் | தமிழ் |
|----------|-------|
| Abacus | கணக்குச்சட்டம் |
| Algorithm | படிமுறை |
```

### 7. அகராதி அல்லாத ஆவணங்களுக்கு (non-glossary documents) பயன்படுத்த முடியுமா?

ஆம், ஓரளவிற்கு. `pdf_to_unicode()` செயல்பாடு (function) எந்த TSCII PDF-லிருந்தும் Unicode உரையை பிரித்தெடுக்கும். ஆனால், அட்டவணை வடிவமைப்பு (table formatting) மட்டும் `ஆங்கிலம் - தமிழ்` என்ற வடிவத்திற்கே பொருந்தும். மற்ற ஆவணங்களுக்கு, நிரலை சிறிது மாற்றி raw Unicode உரையை நேரடியாகப் பயன்படுத்தலாம்.

---

### 8. "No module named 'tamil'" என்ற பிழை வந்தால் என்ன செய்வது?

`open-tamil` நூலகம் சரியாக நிறுவப்படவில்லை என்று பொருள். பின்வரும் கட்டளையை இயக்கவும்:

```bash
pip install open-tamil
```

Python virtual environment பயன்படுத்துகிறீர்களா என்று கவனியுங்கள் — `pip` சரியான சூழலில் (environment) இயங்குகிறதா என்று சரிபார்க்கவும்.

---

### 9. வெளியீட்டில் சில வரிகள் தவறாக மாற்றப்பட்டுள்ளன — ஏன்?

TSCII-PDF-ல் உள்ள சில எழுத்துருக்கள் (fonts) நிலையான TSCII-யைப் பின்பற்றாமல், சற்று மாறுபட்ட குறியாக்கம் கொண்டிருக்கலாம். அப்போது சில எழுத்துக்கள் சரியாக மாறாமல் போகலாம். இது PDF-ல் உள்ள மூல எழுத்துரு (font) சார்ந்த வரம்பு ஆகும்.
